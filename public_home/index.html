Everything you need

Built to run your community

From casual hangouts to professional studios — NexusGuild scales with you.

💬
Real-Time Messaging
Instant message delivery via WebSocket — with typing indicators, replies, pinned messages, rich embeds, and zero perceived lag.
🔊
Crystal-Clear Voice
Powered by LiveKit, NexusGuild delivers low-latency voice channels and DM calls — with per-user mic and deafen controls baked in.
🔐
Granular Permissions
A full bitwise permissions system — set roles, channel overrides, and member-level rules with surgical precision. Server owners always retain control.
🤖
Bot Ecosystem
A REST + gateway bot API lets developers build powerful integrations and automations. Register slash commands, receive real-time gateway events, and respond to interactions.
📋
Forum Channels
Structured discussion without the chaos. Create threaded forum posts with titles, replies, and attachments — perfect for support, ideas, or deep dives.
🎲
Virtual Tabletop
A built-in VTT channel type for TTRPGs and game masters — run sessions directly within your community, no external tools required.
🧵
Threads & Reactions
Branch conversations into threads without leaving the channel. React with Unicode emoji or your own server's custom emojis — including cross-server support.
📁
Rich Media Sharing
Multi-file uploads, automatic link previews with OG metadata, YouTube and Twitch embed support, and a dedicated media channel type.
📬
DMs & Group Chats
1-on-1 and group direct messages with full feature parity — reactions, file attachments, voice calls, and member management.
